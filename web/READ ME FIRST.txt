INSPECTIT.APP — HOW THIS FOLDER WORKS
=====================================
Last organized: 2026-06-24

THE ONE FILE THAT MATTERS
-------------------------
  inspectit-app.html   <-- THIS is the live app. The Desktop
                           "Vehicle Inspection" app opens this exact file.

The Desktop launcher always opens a file named "inspectit-app.html"
(no version number). So the filename NEVER changes — only its contents do.


HOW TO UPDATE TO A NEW VERSION (the only rule to remember)
----------------------------------------------------------
When you get a newer version of the app (e.g. you download a new
.html from a chat into your Downloads folder):

  1. Move the OLD inspectit-app.html into the "archive (old versions)"
     folder (rename it with today's date so you know which is which).
  2. Put the NEW file in this folder and rename it to exactly:
        inspectit-app.html
  3. Double-click the Desktop app — it now runs the new version.

That's it. Because the launcher always opens "inspectit-app.html",
you never have to touch the Desktop app again.

(Or just ask Claude Code to "update inspectit to the new version in my
Downloads" and it will do steps 1-3 for you.)

IMPORTANT
---------
- Your inspection DATA (vehicles, past inspections, repair tickets) is
  saved inside your web browser, NOT in these files. Don't switch
  browsers or clear browser data, or you can lose your records.
- Keep this whole folder where it is. The Desktop app points to this
  exact location.
